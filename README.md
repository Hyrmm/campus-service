## :triangular_flag_on_post:前言

此项目为在校的一个公益性项目，主要针对在校大学生的一个个服务平台，该平台主要解决在校学生的生活交流。 例如：失物招领、闲置物品交易、活动（地理）打卡签到、朋友圈、校内公告发布、在线聊天等等。

此项目主要使用元素`微信小程序`，UI使用的的`VantUi`
